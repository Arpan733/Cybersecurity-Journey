What happens when you visit a website?
When I press enter on website, my system will look for an IP address for that domain. To know that IP address, my system will contact the connected router using a switch. Router will look in temporary DNS table, if it founds it then it will return otherwise, router will find to the DNS Server using the internet with an ARP protocol. After getting the address of DNS, router will ask to resolve the website name to IP address. After getting IP address of the website, it will generate a query to get files to generate the webpage. The query will send to the router again using a switch. The router will connect to the website server through the internet and establish an TCP/UDP protocol to fetch all the files. Once getting all the files, my system will load the webpage for that website.
