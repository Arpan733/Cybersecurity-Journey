# Writeups

This folder contains writeups for labs, challenges, and security exercises.

## Categories

- TryHackMe
- Hack The Box
- Capture The Flag (CTF)
- Vulnerability Analysis
- Incident Investigation

## Writeup Structure

Each writeup should include:

1. Objective
2. Environment
3. Methodology
4. Findings
5. Lessons Learned

The goal is to develop technical documentation and reporting skills while tracking progress.
